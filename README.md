# C++ Programming Basics & Fundamentals

A comprehensive collection of C++ programs and practical exercises covering fundamental concepts, programming logic, operations, and conditional structures.

## 📚 Overview
This repository serves as a structured portfolio demonstrating core programming principles in C++. It ranges from basic variables and input/output processing to mathematical algorithms, logical operations, and decision-making workflows.

## 🔍 Key Concepts Covered

### 1. Variables, Data Input & Output (I/O)
* **Basic Console I/O:** Reading user inputs using `cin` and printing formatted outputs using `cout` (e.g., `AhlamC++12.cpp` and `AhlamC++13.cpp`).
* **Static Assignments:** Programs featuring predefined variable states to demonstrate output verification.

### 2. Basic Arithmetic & Geometric Calculations
* **Area & Perimeter Algorithms:** Custom scripts calculating geometric formulas, including:
  * **Rectangle Area:** Computing area based on `height` and `width` values (e.g., `AhlamC++2.cpp`, `AhlamC++14.cpp`).
  * **Circle Area:** Utilizing float types and constant approximations like $\pi = 3.14$ (e.g., `AhlamC++3.cpp`).
  * **Perimeter (Moheet):** Basic multiplication and geometric property calculation.

### 3. Increment & Decrement Operators (`++`, `--`)
* **Prefix vs. Postfix Analysis:** A dedicated set of examples analyzing the critical behavioral differences between pre-increment (`++x`) and post-increment (`x++`) within execution lines (e.g., `AhlamC++1.cpp`, `AhlamC++15.cpp`).

### 4. Conditional Statements (`if-else`)
* **Number Properties:** Automated logical checks for positive/negative values or even/odd classifications.
* **Score Evaluation:** A grading workflow utilizing compound `else if` ladders to convert numerical scores into formal letter grades (A through F).
* **Absolute Value:** Simple logic implementations returning the non-negative magnitude of an integer.

### 5. Selection Structures (`switch-case`)
* **Interactive Calculator:** Taking dual operands and a functional char sign (`+`, `-`, `*`) to route math operations.
* **Default Case Handling:** Validating structural data inputs (like gender parameters) to log accurate information or catch unexpected input errors.
